Jung

A high-performance geospatial symbology and cartographic rendering engine written in Rust.

Rust WebAssembly Mapbox GL Compatible MIT / Apache-2.0

Features

Mapbox GL Compatible

Parse and render using the industry-standard Mapbox GL JSON style specification. Drop-in replacement for your existing styles.

WebAssembly Ready

Run the full rendering engine in the browser. Style and render features client-side with native performance.

High Performance

Pure Rust with zero-copy geometry processing. Designed for rendering millions of features into tiles at scale.

Extensible Symbology

Beyond basic fill/stroke: graduated symbols, proportional sizing, label placement, and military symbology (MIL-STD-2525).

Multiple Outputs

Render to raw RGBA pixels for tile generation, SVG for print/export, or vector draw commands for custom backends.

TileTopia Ecosystem

Integrates with TileTopia for tile pipelines, ViewTopia for visualization, and Ptolemy for versioned geodata.

Quick Start

use jung_core::renderer::{BBox, Renderer};
use jung_style::parse_style;

let style = parse_style(r#"{
    "layers": [{
        "id": "cities",
        "paint": { "circle-color": "#ff0000", "circle-radius": 5.0 }
    }]
}"#).unwrap();

let renderer = Renderer::new(512, 512).unwrap();
let bbox = BBox { min_x: -180.0, min_y: -90.0, max_x: 180.0, max_y: 90.0 };
let pixels = renderer.render(&style, &features, &bbox).unwrap();
